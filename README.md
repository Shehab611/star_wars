<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the Repo</p>

```
git clone https://github.com/Shehab611/star_wars.git
```

<p>2. Go to directory</p>

```
cd star_wars
```

<p>3. Open on VsCode</p>

```
code .
```

<p>4. Get dependencies</p>

```
flutter pub get
```

<p>5. Run The App</p>

```
flutter run
```
